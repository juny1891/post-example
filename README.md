# 콘솔 게시판 애플리케이션

이 프로젝트는 Node.js를 사용하여 간단한 콘솔 기반 게시판 애플리케이션을 구현한 것입니다. 이 애플리케이션을 통해 사용자는 게시글을 생성, 조회, 수정 및 삭제할 수 있습니다.

##  시작하기 전에
이 애플리케이션을 실행하기 위해서는 Node.js와 IDE가 필요합니다. 아래 링크를 통해 Node.js와 Visual Studio Code를 설치할 수 있습니다.

❋ Node 버전은 18.11.0을 설치하였습니다. 

참고 : https://engineeringcode.tistory.com/315

### readline 모듈에 대한 설명
Node.js의 readline 모듈은 사용자로부터 명령 줄 입력을 읽고 처리하는 데 사용됩니다. 이 모듈은 인터랙티브 명령 줄 인터페이스를 생성하고 관리하는 기능을 제공합니다. 본 프로젝트에서 readline 모듈은 사용자의 입력을 받아 게시글을 생성, 조회, 수정, 및 삭제하는 데 필요한 정보를 얻는 데 사용됩니다.

### readline의 동작 방식
본 애플리케이션에서 readline 인터페이스는 다음과 같은 방식으로 동작합니다:

1. 애플리케이션 실행 시, readline 인터페이스가 생성됩니다.
2. 사용자에게 특정 입력(예: 게시글 작성, 수정, 삭제 등)을 요청하는 프롬프트가 표시됩니다.
3. 사용자가 입력을 완료하고 Enter 키를 누르면, 해당 입력이 애플리케이션으로 전송됩니다.
4. 애플리케이션은 이 입력을 받아 해당하는 기능(게시글 작성, 조회 등)을 수행합니다.
5. 사용자가 애플리케이션을 종료하려고 할 때까지 이 과정이 반복됩니다.

## 설치 방법
1. 이 프로젝트를 클론하거나 다운로드합니다.
2. 프로젝트 디렉토리로 이동합니다.
3. 아래의 명령어를 실행하여 애플리케이션을 시작합니다.

```bash
node main.js
```

## 사용 방법 및 예시
```bash
게시판 관리 (1: 조회, 2: 작성, 3: 수정, 4: 삭제, 5: 종료) 선택:
```

### 1. 게시글 조회
- 입력: 1 입력
- 결과: 현재 저장된 모든 게시글의 제목과 내용이 표시됩니다.
```bash
게시판 관리 (1: 조회, 2: 작성, 3: 수정, 4: 삭제, 5: 종료) 선택: 1
게시판 목록:
0: 제목: 테스트2, 내용: 우아
```
### 2. 새 게시글 작성
- 입력: 2 입력 후, 제목과 내용을 입력합니다. (내용을 입력 후 Enter를 누르면 게시글이 생성되며, "게시글이 생성되었습니다."라는 메시지가 표시됩니다.)
- 예시:
```bash
게시판 관리 (1: 조회, 2: 작성, 3: 수정, 4: 삭제, 5: 종료) 선택: 2
제목: 첫 번째 게시글
내용: 첫 번째 게시글의 내용입니다.
게시글이 생성되었습니다.
```

### 3. 기존 게시글 수정
- 입력: 3 입력 후, 수정할 게시글의 번호, 새 제목, 새 내용을 입력합니다. (내용을 입력 후 Enter를 누르면 게시글이 수정되며, "게시글이 수정되었습니다."라는 메시지가 표시됩니다.)
- 예시:
```bash
게시판 관리 (1: 조회, 2: 작성, 3: 수정, 4: 삭제, 5: 종료) 선택: 3
수정할 게시글 번호: 0
제목: 수정된 게시글
내용: 수정된 내용입니다.
게시글이 수정되었습니다.
```

### 4. 게시글 삭제
- 입력: 4 후, 삭제할 게시글의 번호를 입력합니다. (게시글이 삭제되며, "게시글이 삭제되었습니다."라는 메시지가 표시됩니다.)
- 예시: 
```bash
게시판 관리 (1: 조회, 2: 작성, 3: 수정, 4: 삭제, 5: 종료) 선택: 4
삭제할 게시글 번호: 0
게시글이 삭제되었습니다.
```
   

### 5. 애플리케이션 종료
- 입력: 5 입력 후, Enter를 누릅니다. 
- 결과: 프로그램이 종료됩니다.
